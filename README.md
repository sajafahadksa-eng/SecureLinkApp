# SecureLink AI

SecureLink AI is a prototype security platform designed to solve a major problem in modern security systems: the lack of integration between different security technologies.

Many organizations use multiple security tools such as surveillance cameras, access control systems, and network monitoring solutions. However, these systems often work independently, which can delay threat detection and make it harder to understand the full security situation.

SecureLink AI provides a unified dashboard that collects and analyzes events from multiple sources and calculates a Threat Score to help security teams quickly identify potential risks.

## Features

- Unified Threat Score based on multiple events
- Simulation of security events
- Visual risk indicator
- Security event logging
- Simple dashboard for monitoring threats

## Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn

## Future Improvements

- Integration with real security systems
- Advanced AI-based threat analysis
